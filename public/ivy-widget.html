Ivy PPT Tool

从 PPT 模板和业务提示词出发，先生成图片样例，再把确认后的样例图复刻为高质量、可编辑的 PPTX。

ChatGPT App 开发版

工作流程

1 上传模板与需求

在对话中上传 PPT 模板，并描述主题、受众、页数和希望呈现的风格。

2 生成图片样例

调用工具生成生图 prompt，先得到视觉样例，不满意可以继续调整提示词。

3 复刻为可编辑 PPT

确认样例后，将图片拆解为文本、形状、表格、图表和透明素材重建。

在 ChatGPT 中使用

  1. 先运行 ivy_start_workflow 打开工作流。
  2. 提供 PPT 模板和需求后，运行 ivy_build_image_prompt。
  3. 样例图确认后，运行 ivy_build_replication_prompt 准备复刻。

模型能力由使用者自己的 ChatGPT 会话提供。本应用不内置站长的 OpenAI API Key，也不消耗站长的 ChatGPT 额度。

模板驱动 保留模板的版式、配色和商务气质。
可编辑优先 文本和简单图形尽量使用 PPT 原生对象。

下一步工具

当前版本已经提供 ChatGPT App 的 UI 和 prompt 工具。下一版可以继续接入云端 PPTX 渲染器，让用户上传确认后的图片并直接下载可编辑 PPTX。
